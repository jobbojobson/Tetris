"Javascript Tetris".. if that's a name. Andrew Jobson August/September 2019

This version of Tetris tries to be as faithful as possible to the Nintendo Game Boy version of the game in terms of game behaviour and mechanics.

On mobile, landscape mode is recommended

Desktop controls:

  • Move left: left key
  • Move right: right key
  • Soft drop: down key
  • Rotate left: Z key
  • Rotate right: X key
